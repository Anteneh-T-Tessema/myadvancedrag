How HyDE Works
1
Short Query Received
User submits a terse, ambiguous, or highly technical query that may not share vocabulary with your document corpus.

2
LLM Generates Hypothetical Answer
A fast local model (Ollama) generates a structurally plausible document passage — even if factually uncertain, it uses correct jargon and syntax.
→
3
Embed Hypothetical Document
The hypothetical document — not the original query — is embedded. This creates a paragraph-to-paragraph comparison in vector space.
→
4
Superior Retrieval
Recall@K improves significantly because the search vector now structurally matches target document embeddings.
